# FFXIV ACT 科技 / 触发器写作入门指南

**作者：阿洛（MnFeN）**

本仓库包含 ACT 中基础解析插件及触发器的介绍。

主要受众为在 ACT 中使用和创作触发器（以 PvE 战斗内容为主）的开发者，目前正在长期施工中。

如果觉得有语言、表述、理解、使用等任何方面的问题，欢迎在上方随意提交 Issue，或直接 Fork 仓库并修改后提交 PR。
